# Assets Folder

This folder is for optional assets to enhance your proposal website.

## Music File

Add your romantic background music here:

- **File name**: `romantic-music.mp3`
- **Format**: MP3 recommended
- **Duration**: 2-5 minutes (will loop)
- **Volume**: Ensure it's not too loud

### Where to find music:
1. **Free royalty-free sites**:
   - Pixabay Music
   - YouTube Audio Library
   - Freesound.org
   - Bensound

2. **Recommended search terms**:
   - "romantic piano"
   - "soft instrumental"
   - "love theme"
   - "gentle background music"

### Note about music:
- Due to browser security policies, music may require user interaction to start
- The website will work perfectly without music
- Music is optional but adds a nice emotional touch

## Photos (Optional)

You can also add photos here if you want to create a photo gallery on the success screen:

- Create a `photos` subfolder
- Add your favorite couple photos
- Modify the JavaScript to display them

Remember to keep file sizes reasonable for fast loading!
